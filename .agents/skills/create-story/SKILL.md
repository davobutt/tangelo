# Objective
Create a new user story in the backlog, including technical tasks, test scenarios, and integration into the appropriate epic.

# Output
Create user stories in the `backlog` folder with the following structure:
```
backlog/
  epic-name/
    story-name.md
```
Each story file should include:
- **Title**: A concise title for the story.
- **Description**: A detailed description of the story, including the user need and the proposed solution.
- **Acceptance Criteria**: A checklist of conditions that must be met for the story to be considered complete.
